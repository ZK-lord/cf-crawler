# Codeforces Crawler 使用说明

## 环境要求

- Python 3.x（服务器模式需要）
- 能访问 `codeforces.com` 的网络

## 使用方法

### 方式一：一键启动（推荐）

双击 `启动服务器.bat`，浏览器自动打开 `http://localhost:8088`，在页面输入 CF 用户名点添加即可。服务器在后台自动抓取并生成完整报告。

### 方式二：直接查看已有报告

浏览器打开 `output/index.html`，查看预生成的报告。**注意：此方式只能看已有数据，无法添加新用户。**（添加用户功能依赖服务器，直接打开 HTML 文件会报错）

### 方式三：命令行抓取

```bash
# 1. 编辑 sample_users.txt，每行写一个 CF 用户名
# 2. 运行
bin\cf_crawler.exe sample_users.txt
# 3. 浏览器打开 output/index.html
```

### 方式四：源码编译

```bash
make && bin\cf_crawler.exe sample_users.txt
```

## 常见问题

**Q: 页面里添加用户没反应？**

A: 检查三点：① 是否通过 `启动服务器.bat` 访问（不是直接双击 index.html）；② 是否装了 Python；③ 网络能不能连 `codeforces.com`。

**Q: 双击 bat 闪退？**

A: 没装 Python，或者 Python 没加到系统 PATH。装完 Python 后重新双击。

**Q: 提示缺少 DLL？**

A: 确保 `bin/` 下所有 `.dll` 与 `cf_crawler.exe` 在同一个目录。
